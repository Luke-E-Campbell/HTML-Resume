About Me

Hello! I'm Luke Campbell and I love programming.

Introduction:

Throughout the years I've gained a passion for coding through different courses and various
projects. What drove me to becoming a full stack developer was the job growth potential and
the ability to work with teams in order to create fully functional websites, applications, and

Education:

I graduated from Howard Community College in 2019 with an associates degree in
Cybersecurity. During that time I had the pleasure of working hands on with cisco routers,
switches and firewalls, learning the fundamentals of network security, setting up networks,
learning encryption standards, ethical hacking, and Linux system administration. I then graduated
from University of Maryland, Baltimore County (UMBC) in 2022 with a bachelor's degree in
Information Systems. There I gained my passion for full stack development taking courses such
as UX/UI Design, Java Programming I, Java Programming II, Gui Systems Using Java, Database
Application Development and Structured System Analysis/Design.

Skills and Expertise:

Since October of 2023 I've been spending my time learning the ins and outs of web development,

JavaScript, Bootstrap Framework, jQuery library, Node JS runtime environment, Express JS
framework, and I am still in the process of learning more like EJS, PostgreSQL, Python, Java,

Projects:

HTML Portfolio Website:

While learning HTML I created an html only resume portfolio using VSCode. Creating
this resume taught me the basics of html such as the different tags, how to add photos, and
anchors for an "about me" and "contact me" page.

TinDog Website:

While learning CSS, Bootstrap, Flexbox, Grid, I created a dating website for dogs in using

Dice Game Challenge:

While learning about JavaScript and the Docuement Object Model (DOM), I created a dice
game using VSCode where you and a second player go against eachother to see who
has the highest number. If you win the game the game displays that the player has won but if
the player loses then a player lost message is displayed. If the game is a tie then a tie message
is displayed.

Drum Kit Project:

While learning about advanced JavaScript and the Docuement Object Model (DOM), I created
a drum kit using VSCode where based on which image you click it will produce the
sound of that image. In addition I also added the function to used the W, A, S, D, J, K, L keys
that will also play the sound of the image based on which key it is matched up with.

Simon Says Game Challenge:

While learning about jQuery, I created a Simon Says game that keeps track of the games
pattern and the players pattern. The player can press any key to start and everytime the player
gets the right pattern then they go up one level. If the player gets the wrong pattern when
matched to the games pattern then the game ends and the player has the option of starting over.
